# VSCode-Guide shortcuts
Given you use english keyboard.

## Go to line
```
Ctrl + G --> line nr
```

### Highlight a word
```
Ctrl + D
```

### Highlight a line
```
Ctrl + L
```

If you want to edit the same word in multiple places in the same file just press "Ctrl + D" again and again.


## Delete line 
```
Ctrl + X
```

## Comment out code
```
Ctrl + /
```

## Move line up and down 
```
Alt up or Alt down
```

# Navigation related

## Toggle explorer
```
Ctrl + B
```

## Open terminal
```
Ctrl + `
```
