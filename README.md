# VSCode-Guide shortcuts
Given you use english keyboard.

# Editor shortcuts
Go to line
```
Ctrl + G --> line nr
```

Go to method
```
Ctrl + p + @
```

### Highlight a word
```
Ctrl + D
```

### Highlight a line
```
Ctrl + L
```

If you want to edit the same word in multiple places in the same file just press "Ctrl + D" again and again.


## Delete line 
```
Ctrl + X
```

## Comment out code
```
Ctrl + /
```

## Move line up and down 
```
Alt up or Alt down
```

## Select multiple lines
For each line
```
Option + click
```

# Navigation related
